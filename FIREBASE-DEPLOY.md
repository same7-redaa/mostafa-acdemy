# نشر المشروع على Firebase

اقرأ ملف DEPLOYMENT.md للتعليمات الكاملة.

## سريعاً:

```bash
# 1. Build
npm run build

# 2. Deploy (إذا كان Firebase مُعد)
firebase deploy

# أو للمرة الأولى:
firebase init
firebase deploy
```
